Personal productivity app with 5 tools: **Notes**, **Habits**, **Calendar**, **Calculator**, and **Life Tracker**.

Built with Flutter + Material 3, sqflite, Provider, and flutter_local_notifications.

### Features
- **Notes** — color-coded notes with search, tags, pin
- **Habits** — daily tracking with streaks, weekly/monthly views, reminders
- **Calendar** — event management, category colors, day detail with habit/note history
- **Calculator** — standard + scientific (sin/cos/tan/log/sqrt), memory functions, history
- **Life Tracker** — live-updating time elapsed since birth, life progress meter
